Tutorials
See where to change things

Changing the Logo

Changing the logo is very easy. Just replace the file /img/logo.png with yours. If you want to change the filename, you have to replace the name of the file in the /index.html. Simply go to <div class="logo"> (Line: 14) and change the img src to your filename.

Changing the Site Name

Just change the name in the <title> (Line: 6) tag.
Go up

Changing the Background

Simply change the /img/background.jpg file with a new file. Notice: the minimum width should be 900 px. A height of more than 3000 px is recommended. The heigher, the better the effect.
Go up

Changing the Hidden Content Page

Put all your content (no matter if div, p, h1-h6, img, et cetera) into the <article class="moreinformation"> (Line: 70) container. Make sure you do not remove this article or change the class of it. Otherwise the overlay functionality will not work anymore.
Go up

Changing the Google Maps stuff

First of all you have to get a Google Maps API key (https://developers.google.com/maps/documentation/javascript/tutorial#api_key). Then you have to open /index.html and go to Line 8. Replace YOUR_API_KEY and TRUE_OR_FALSE with your API key and configuration.

Open the JavaScript /js/script.js and simply change Line 114 to adjust your Google Maps center. You just have to change the GPS coordinates to your own ones (var yourLocation = new google.maps.LatLng(52.520007, 13.404954);)
Of course you can adjust Google Maps as you want it.

To get your GPS coordinates, you might use this tool: http://itouchmap.com/latlong.html

Go up

Changing the Portfolio images

To change the portfolio images, go to <article class="work"> (Line: 27) and just edit the <img src="img/dummy-portfolio.png" alt="Placeholder"> with your new portfolio image names (in src) and alt-tags for SEO (in alt). Please make sure your images have a size of 200 x 200 Pixels.
Go up

Changing the Employee images

To change the employee images, go to <article class="about"> (Line: 36) and just edit the <img src="img/dummy-employee.png" alt="Placeholder"> with your new portfolio image names (in src) and alt-tags for SEO (in alt). Please make sure your images have a size of 80 x 80 Pixels. Greyscale is added automatically.

To add a completely new employee, just copy and paste another <div class="employee clearfix">.
Go up

Changing the Contact mails

You can change the "hire us" and "contact us" e-mails and subjects. Go to line 56 and 60 and change the href-attribute of the a-tags. <a href="mailto:test@mail.com" class="button mi">
test@mail.com shows where the email will be send to. Just change this to your mail(s).
